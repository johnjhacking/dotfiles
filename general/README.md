### Shift Arrow Navigation
To use the shift arrow navigation in Terminal.app:

1. Terminal > Preferences > Settings > Keyboard
2. Click +
	* Key: Cursor Up
	* Modifier: Shift
	* Action: Send Text
	* Enter: \033[1;2A
	* Click OK
3. Click +
	* Key: Cursor Down
	* Modifier: Shift
	* Action: Send Text
	* Enter: \033[1;2B
	* Click OK
